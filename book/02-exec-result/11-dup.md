# ここでは，dupシステムコールの使い方について説明します．

これらのシステムコールは、ファイルディスクリプタを複製するために使用されます。複製されたファイルディスクリプタは、元のファイルディスクリプタと同じファイルにアクセスできます。

### プロトタイプ

ライブラリ関数のプロトタイプは以下のようになっています．

```c
int dup(int oldfd);
int dup2(int oldfd, int newfd);
```

### 概要

`dup`システムコールは，ファイルディスクリプタを複製します．あるファイルディスクリプタを使ってあるファイルにアクセスできるとき，このシステムコールを使ってファイルディスクリプタを複製すると，新しいファイルディスクリプタを使ってそのファイルにアクセスできるようになります．

### ファイルの状態の共有

ファイルの参照位置（オフセット）やオープンモードなど、開いているファイルそのものに紐づく状態は、複製されたファイルディスクリプタと元のファイルディスクリプタで共有されます。一方、close-on-execフラグのようにファイルディスクリプタ自体に紐づく状態は共有されません。

### ファイルディスクリプタのクローズ

複製されたファイルディスクリプタまたは元のファイルディスクリプタのいずれかを閉じても、もう一方のファイルディスクリプタには影響しません。

### 引数

`dup`の引数は複製元のファイルディスクリプタです． `dup2`の第一引数は複製元のファイルディスクリプタで，第二引数は新しいファイルディスクリプタです．第二引数として指定した番号が新しいファイルディスクリプタになります．

### 返り値

成功した場合，新しいファイルディスクリプタです．失敗した場合，`-1`が返ります．

### 使用方法

標準出力のファイルディスクリプタを複製するには以下のように記述します．

```c
int fd = dup(STDOUT_FILENO);
```

`123`というファイルディスクリプタで標準出力に書き込めるようにするには以下のように記述します．

```c
dup2(STDOUT_FILENO,123);
```

## 例

複製したファイルディスクリプタを利用して標準出力に書き込みます．

```c
#include <unistd.h>
#include <stdio.h>

int main() {
    int new_fd = dup(STDOUT_FILENO);
    if (new_fd == -1) {
        perror("dup failed");
        return 1;
    }
    write(new_fd, "Hello\n", 6);
    close(new_fd);
    return 0;
}
```

この例では、**`dup`**関数を使用して、標準出力ファイルディスクリプタ（**`STDOUT_FILENO`**）を複製し、新しいファイルディスクリプタ**`new_fd`**に割り当てています。その後、**`write`**関数を使用して、新しいファイルディスクリプタを使って標準出力に**`"Hello\n"`**という文字列を書き込んでいます。このプログラムを実行すると、標準出力に**`"Hello"`**と表示されます。

完全なサンプルコードは [examples/02-exec-result/dup](../../examples/02-exec-result/dup/README.md) にあります。
