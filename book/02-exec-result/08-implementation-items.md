# 実装していく項目

ここでは，LinuxでのC言語の基礎の理解度の確認のため，以下のような関数を実装します．

- 末尾がヌル文字となっている文字列のポインタを受け取ります
- 文字列はASCII文字のみを含みます
- その文字列のヌル文字までの部分を標準出力に出力します
- writeシステムコールを用います
- `stdio.h`や`string.h`，`stdlib.h`などの標準ライブラリを用いません

## 仕様

### プロトタイプ

```c
int print_string(const char* const str);
```

### 返り値

成功した場合，標準出力に書き込んだバイト数．失敗した場合，`-1`

### 引数

`str`はヌル文字を末尾にもつ文字列のポインタが与えられる．

### 挙動

`str`に指定された文字列のヌル文字までの部分を標準出力に書き込んだ後，改行文字`'\n'`も標準出力に書き込む．返り値のバイト数には，最後に書き込む改行文字も含める．

```c
int main(){
    print_string("Hello, World!");
}
```

## hints

まず，`for`文などを用いて文字列の長さを調べて，その後で`write`システムコールを使用します．
